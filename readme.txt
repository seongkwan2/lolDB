팀프로젝트 git 사용방법

1. 각자 만든것 각자의 브런치에서 git push -u origin 브런치명

2. 마스터로 이동후 git merge sk (각자 테이블명)

받을때
1. git checkout master에서 github페이지에있는 데이터들을 git pull로 받음

2. git merge sk

3. 겹치는것들 수정 저장

4. add , commit, push
